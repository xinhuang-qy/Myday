# 二、具体奖励机制（可直接用）

## 1. 局内即时奖励（爽感来源）

- 下落过程中收集**情绪碎片 / 光粒**
- 每捡一个都有轻微震动 + 闪光
- 结算时按碎片数量给**诊疗费**

## 2. 通关奖励（核心动力）

治愈一个病人 = 获得：

- 诊疗费（金币）
- 病人感谢信（可展示在大厅）
- 概率解锁：新皮肤 / 新特效

## 3. 长期成长奖励（让玩家一直玩）

### ① 治疗师等级

- 等级提升 → 血量上限 + 1 / 精神攻击变强
- 等级高了外观会更精致（衣服变好看）

### ② 外观皮肤（最吸引大众）

- 医生不同形象：温柔医生、酷飒医生、可爱风
- 树洞皮肤：樱花树洞、星空树洞、荧光树洞
- 攻击特效：星光、花瓣、音符、气泡

### ③ 病人图鉴收集

每个病人治愈后会出现在**心灵之树大厅**

可以看他们的小故事、感谢语

→ 满足收集欲，非常适合休闲玩家

### ④ 成就奖励

- 下 10/50/100 层
- 无伤通关
- 治愈 10 个病人
- 收集全部情绪碎片
    
    完成就给金币 / 限定皮肤

## 4. 每日轻量奖励（拉留存）

- 每日登录 → 少量金币
- 每日 3 次免费 “诊疗任务”
- 完成就给奖励，不逼肝
