# 1. 生成视频的形式

## 1. 提示词
```
参考提供的角色图片，生成Q版日系平涂卡通风角色的走路动画序列帧。

【风格约束】

- Q版比例（头身比约 1:3）
    
- 日系平涂卡通风
    
- 颜色分块清晰，阴影为1级
    
- 线条干净稳定
    
- 保持原角色设计完全一致（发型、服装、配色）
    

【基础规格约束】

- 正视图
    
- 8帧循环动画（loop），横向排成一列
    
- 每帧尺寸一致
    
- 背景为纯绿色
    
- 无动态模糊、无镜头移动、无透视变化

    

布局与网格（核心约束）

- 每一帧占据固定矩形区域
    
- 每帧之间固定间距：50像素
    
- 间距为纯背景区域，不包含角色
    
- 所有帧严格对齐统一网格
    
- 每帧 bounding box 完全一致
    
- 网格仅用于对齐，不可见，透明
    

【角色位置约束（防止误判为位移）】

- 每一帧中角色中心点X坐标完全一致
    
- 水平位移 = 0像素
    
- 角色始终位于各自帧的中心区域
    

【整体运动规律】

- 身体上下浮动：Frame1最低，Frame4最高，总高度差为3像素
    
- 上升节奏：每帧 +1像素（Frame1→Frame4）
    
- 下降：Frame4 → Frame5 直接回落
    
- 头部仅随身体上下移动，不产生左右偏移（≤1像素）
    
- 手臂与腿部反向摆动（幅度略夸张）
    
- 重心在左右脚之间交替转移
    
- 动作具有轻微弹性（bounce）
    

【逐帧动作约束】

Frame1（接触 / 右脚前）：

- 右腿：前摆约75°，脚跟着地
    
- 左腿：后摆约15°，脚尖点地（超过身体中线）
    
- 右臂：后摆约20°
    
- 左臂：前摆约50°
    
- 身体高度：0px（最低点）
    

Frame2（承重）：

- 左腿：开始前抬（约50°）
    
- 右腿：承重，小腿趋近垂直
    
- 手臂：延续摆动趋势（继续后摆 / 开始回收）
    
- 身体高度：+1px
    

Frame3（经过位 passing）：

- 左腿：经过身体中线（约85°）
    
- 双腿接近交叉
    
- 手臂：穿越中线反向摆动
    
- 身体高度：+2px
    

Frame4（最高点）：

- 左腿：前摆极限（约95°）
    
- 右腿：脚跟抬起
    
- 手臂摆动达到最大幅度
    
- 身体高度：+3px（最高点）
    

Frame5（接触 / 左脚前）：

- 动作镜像 Frame1
    
- 身体高度：0px
    

Frame6（承重）：

- 动作镜像 Frame2
    
- 身体高度：+1px
    

Frame7（经过位）：

- 动作镜像 Frame3
    
- 身体高度：+2px
    

Frame8（过渡帧）：

- 接近Frame1但未接触
    
- 身体高度：+1px
    
- 准备无缝循环
    

【关键一致性约束】

- 头部大小、五官位置在所有帧中完全一致
    
- 身体比例不变（禁止拉伸或压缩）
    
- 相邻帧轮廓变化平滑
    
- 不允许出现结构漂移或细节丢失
```
